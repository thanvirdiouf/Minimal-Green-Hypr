# Minimal Green Hypr

My Arch Linux dotfiles for a minimal, green-themed Hyprland environment. 

> **Note:** This repository is currently a work in progress. Additional program configurations and setup scripts will be added later.

## Included Configurations

This repository currently includes configuration files for the following:

* **[Hyprland](hypr/)**: The core Wayland compositor.
* **[Waybar](waybar/)**: Status bar.
* **[Kitty](kitty/)**: Terminal emulator.
* **[Dunst](dunst/)**: Notification daemon.
